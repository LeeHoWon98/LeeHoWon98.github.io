## 시작하며

오늘부터 GitHub Pages를 이용해 기술 블로그를 시작했다.

이 블로그에는 C언어, 파이썬, 자료구조, 알고리즘, AI 프로젝트 경험을 정리할 예정이다.

## 왜 기술 블로그를 만들었는가?

공부한 내용을 단순히 저장하는 것이 아니라, 나중에 다시 설명할 수 있는 형태로 정리하기 위해서다.
